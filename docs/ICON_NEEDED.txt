⚠️ ACTION REQUIRED: Add Bot Icon

You need to add a bot icon to complete the TON Connect setup.

File name: icon-180x180.png
Location: /docs/icon-180x180.png
Size: 180x180 pixels
Format: PNG

Options:

1. Use your bot's Telegram avatar:
   - Get profile photo from @tonrafflebest_bot
   - Resize to 180x180 pixels
   - Save as docs/icon-180x180.png

2. Create a custom icon:
   - Use Canva, Figma, or similar tool
   - Design a 180x180px PNG icon
   - Save as docs/icon-180x180.png

3. Use a placeholder temporarily:
   Run this command:
   curl -o docs/icon-180x180.png "https://via.placeholder.com/180/667eea/ffffff?text=TON"

After adding the icon:
   git add docs/icon-180x180.png
   git commit -m "Add bot icon for TON Connect"
   git push

Then verify at:
https://f2re.github.io/raffle-telegram-bot/icon-180x180.png
